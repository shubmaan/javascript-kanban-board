# Javascript Kanban Board
A Kanban board application built with HTML, CSS, and JavaScript for organizing tasks through a simple drag-and-drop interface. Tasks can be created, moved between workflow stages, deleted, and are automatically saved using Local Storage, allowing data to persist between browser sessions.

## Features

* Create new tasks with a title and description
* Organize tasks into **To-Do**, **In Progress**, and **Done** columns
* Drag and drop tasks between columns
* Delete tasks
* Automatic task counters for each column
* Local Storage support for data persistence
* Responsive layout for desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How to Use

1. Click **Add New Task**.
2. Enter a task title and description.
3. Add the task to the board.
4. Drag tasks between columns as work progresses.
5. Delete tasks when they are no longer needed.
6. Tasks are automatically saved and restored when the page is reopened.
